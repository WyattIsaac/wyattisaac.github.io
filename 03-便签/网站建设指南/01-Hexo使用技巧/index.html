Hexo 使用技巧

Hexo基础命令使用

这部分内容以 Hexo 官方文档为准，这里记录一些最常用的。传送门：命令 | Hexo

  • 初始化： hexo init && npm install
  • 生成博客：hexo g
  • 部署博客：hexo d
  • 清空缓存：hexo clean
  • 启动本地服务器：hexo s
  • 组合命令：hexo clean && hexo g -d
  • 组合命令：hexo g && hexo s
